1. Асимптотическая сложность алгоритма это:
	а. Функция зависимости объёма работы, которая выполняется некоторым алгоритмом, от
	размера входных данных
	б. Показатель определяющий сложность выполнения программы на языке Си
	в. Общее понятие для всех алгоритмов отвечающее за надёжность выполнения алгоритма
	г. Понятие придуманное для описания различных вариантов решения алгоритма Евклида

Ответ - а

2. О-большое это:
	а. Константа в языке Си
	б. Обозначение математической функции используемое для расчёта памяти занимаемой
	данными обрабатываемыми алгоритмом
	в. Худшее время выполнения алгоритмов
	г. Специальная функция в языке Си дающая возможность определить время выполнения
	программы

Ответ - в

3. Тета-большое(Θ) это:
	а. Среднее время выполнения алгоритмов
	б. Параметр определяющий дельту между началом выполнения программы и её концом
	в. Обозначение математической функции используемое для расчёта минимального
	количества памяти занимаемого данными обрабатываемыми алгоритмом
	г. Специальная функция в языке Си дающая возможность определить максимальное время
	выполнения программы

Ответ - а

4. Омега Большое (Ω) это:
	а. Параметр используемый в базах данных и хранящий в себе общее количество элементов
	для каждой таблицы
	б. Обозначение математической функции используемое для расчёта максимального
	количества памяти занимаемого данными обрабатываемыми алгоритмом
	в. Специальная функция в языке Си дающая возможность определить минимальное время
	выполнения программы
	г. Лучшее время выполнения алгоритмов

Ответ - г

	б. Логарифмическая O(log n)
	в. Линейная O(n)

6. Линейным называется алгоритм:
	а. Который успевает выполниться за чётко определённый такт работы процессора
	б. Имеющий чётко определённую последовательность действий
	в. Запускаемый отдельным потоком в программе
	г. Исполняемый только с использованием ассемблера, т.к. линейные алгоритмы более не
	используются

Ответ - б

7. Ветвящимся называется алгоритм:
	а. Запускающий в процессе своей работы дополнительные потоки
	б. Имеющий в своём коде рекурсивные запуски функций
	в. Имеющий варианты исполнения в процессе выполнения программы
	г. Построенный на основе графов

Ответ - в

8. Циклическим называется алгоритм:
	а. Который в основе имеет периодические календарные вычисления
	б. Рекурсивно выполняющий вычисления
	в. Тестирования программы, запускающийся в процессе её отладки
	г. Запускающий в процессе своей работы повторяющиеся действия на основе циклов

Ответ - г
